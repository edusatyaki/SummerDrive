## Folder guide

| Folder | What goes here |
|--------|----------------|
| `Data Cleaning/` | Load the raw dataset, handle missing values, fix data types, remove duplicates, and export a clean dataset the rest of the team can use. |
| `Data Analysis/` | Explore the clean data, run the statistics, find patterns, and document the key insights that answer the project questions. |
| `Dashboard/` | Build the interactive dashboard and visualizations that present the findings clearly to a business audience. |
| `Report/` | Write the final report that ties cleaning, analysis, and dashboard work into one business story with conclusions and recommendations. |

## Workflow
Data Cleaning -> Data Analysis -> Dashboard -> Report. Each owner picks up the output of the previous step.
