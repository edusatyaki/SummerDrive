## Structure

```
SummerDrive/
  Cohort-N/
    <Project>/
      Data Cleaning/   README.md
      Data Analysis/   README.md
      Dashboard/       README.md
      Report/          README.md
```

## Naming notes
- `A-B Test` is used instead of `A/B Test` because a slash is a path separator and would break the folder.
- Each project uses the same four role folders so cohorts stay comparable, even though the original
  groups used different labels (Statistics, Analytics, Forecasting, Visualization, EDA, Data Prep).
- The Report role is open in every project. Assign owners or remove the folders if not needed.
